git submodule

发表于 2020-05-30 | 分类于 coding

git submodule即git子模块，将一个git仓库作为另一个git仓库的子目录

添加一个子模块

会增加：

  1. .gitmodules文件，记录每个子模块的引用信息
  2. 相应的子模块

克隆一个带有submodules的仓库

修改submodule （在引用项目中修改）

  1. 在这个lib中修改并提交
  2. 在引用这个module的项目中提交
  3. 在主项目中git pull
