解决hexo网站部署到github以后不能显示hexo主题

发表于 2017-08-21 | 分类于 play

hexo搭建好后本地server启动出现正常默认landscape主题样式

但是部署到github以后出现的页面只有文字没有css样式

这是配置文件_config.yml中url和root的设置问题

部署的配置中，最好的方式是把项目的名称和github的名称相同
deploy:
type: git
repo: https://github.com/QQCR/QQCR.github.io.git
branch: master
否则像其他教程中直接进入
https://QQCR.github.io
是不可行的，比如repo的时候设置的名字是blog，域名那个地方输入的时候必须输入
https://QQCR.github.io/blog.github.io
才能进入

确定好这点以后，如果不在子目录中，url注意就是这个网址，
https://QQCR.github.io
而不是随意写的一个域名
